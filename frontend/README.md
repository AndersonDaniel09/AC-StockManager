# Frontend - AC StockManager

Aplicación cliente construida con React + Vite.

## Scripts

- `npm run dev` → ejecutar en desarrollo
- `npm run dev:lan` → ejecutar en red local (`0.0.0.0:5173`)
- `npm run build` → build de producción
- `npm run preview` → previsualizar build
